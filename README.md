## RuparupaMobileAutomation

This source code is used to automate mobile platform of ruparupa.com

## Prerequisites

1. JDK 1.8 or above
2. Apache Maven
3. IDE(Intelij/Eclipse)
4. Git
5. Android studio( for emulator only)

```python
1. turn on your emulator on Android studio
```
